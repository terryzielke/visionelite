== Description ==
Adds a dropdown to the admin top bar where you can quickly click to copy your brand colors. This is a very convenient when working with a page builder tool and need to set colors for a bunch of different elements.

**Features:**

1. admin bar menu
1. click to copy hex color
1. set your own colors in Settings

== Screenshots ==
1. Set Colors Settings.
2. Admin Bar Menu.

== Installation ==
1. Upload plugin folder to the "/wp-content/plugins/" directory.
1. Activate the plugin through the "Plugins" menu in WordPress.
